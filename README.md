# Charp
# Проект по отработке навыков C#

Коллекция практических задач и проектов для закрепления навыков программирования на C#.


## Навыки

C#

## Описание


Этот проект создан для систематической отработки навыков программирования на языке C#. Он включает приложение для работы с шифрованием данных, демонстрирующее применение криптографических алгоритмов в .NET.


Цель проекта:
* закрепить синтаксис C#;
* отработать принципы ООП в C#;
* научиться писать безопасный код для работы с конфиденциальными данными;
* получить опыт работы с инструментами разработки на C#;
* понять различия между симметричным и асимметричным шифрованием.
